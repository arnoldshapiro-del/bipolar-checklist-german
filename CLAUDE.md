# Bipolar Checklist German

## Project Info
- **Purpose**: German-language interactive bipolar spectrum diagnostic guide for primary care physicians
- **GitHub repo**: arnoldshapiro-del/bipolar-checklist-german
- **Netlify URL**: https://bipolar-checklist-german.netlify.app
- **Tech stack**: Single-page HTML/CSS/JavaScript app, jsPDF for PDF generation
- **Branch**: master

## Current Status: COMPLETE
Fully translated from English to German. All 20 tabs, 7 interactive questionnaires, treatment modules, scoring systems, and UI text are in German.

## Features Completed
- Full German translation of all 20 navigation tabs (April 2026)
- 7 interactive questionnaires with real-time scoring:
  1. Signs of Bipolarity Checklist (25 items)
  2. Rapid Mood Screener (RMS, 6 items)
  3. Bipolarity Index (5 domains, 100 points)
  4. DIG FAST (7 items)
  5. Patient Self-Assessment (16 items)
  6. Mixed Features Screen (7 items)
  7. Red Flag Pattern Engine (12 items, weighted scoring)
- Treatment module with 15 sub-tabs
- Life Chart with interactive grid
- Risk Dashboard with SVG ring meters
- Patient Handout Generator
- Clinical Summary Generator
- German emergency numbers (112, Telefonseelsorge: 0800 111 0 111)
- PWA manifest with German name/description

## File Structure
- `index.html` — Complete app (4,398 lines: CSS + HTML + JavaScript)
- `manifest.json` — PWA manifest (German)
